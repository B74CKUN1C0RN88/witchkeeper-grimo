# WitchKeeper / Grimo Website

## Hochladen
Lade alle Dateien dieses Ordners in das Repository `B74CKUN1C0RN88/witchkeeper-grimo` hoch.

## GitHub Pages aktivieren
1. Repository öffnen.
2. **Settings** öffnen.
3. Links **Pages** auswählen.
4. Unter **Build and deployment** bei **Source** „Deploy from a branch“ wählen.
5. Branch **main** und Ordner **/(root)** auswählen.
6. **Save** klicken.

Danach sollten diese Links funktionieren:
- https://b74ckun1c0rn88.github.io/witchkeeper-grimo/
- https://b74ckun1c0rn88.github.io/witchkeeper-grimo/terms.html
