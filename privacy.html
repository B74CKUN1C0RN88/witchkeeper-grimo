2. Verarbeitete Daten

Je nach Funktion können Discord-Benutzer-ID, Anzeigename, Server-, Kanal- und Rollen-IDs, Mitgliedschaftsinformationen sowie Nachrichteninhalte für Übersetzungen verarbeitet werden. Freiwillige Angaben wie Geburtstage werden nur verarbeitet, wenn eine solche Funktion aktiviert wird.

3. Zwecke

Die Daten werden zur Bereitstellung von Verifizierung, Übersetzung, Begrüßung, Moderation und Serververwaltung verarbeitet.

4. Übersetzungsdienst

Bei der Übersetzung kann der Nachrichteninhalt an einen externen Übersetzungsdienst übertragen werden. Vertrauliche Inhalte sollten daher nicht automatisch übersetzt werden.

5. Speicherdauer

Daten werden nur so lange gespeichert, wie dies für die jeweilige Funktion erforderlich ist. Dauerhaft gespeicherte Einstellungen oder freiwillige Angaben bleiben bis zur Löschung oder Deaktivierung gespeichert.

6. Weitergabe

Daten werden nicht verkauft. Eine Übermittlung erfolgt nur, soweit sie technisch für Discord oder einen Übersetzungsdienst erforderlich ist.

7. Rechte der Nutzer

Nutzer können Auskunft, Berichtigung oder Löschung dauerhaft gespeicherter Daten verlangen, soweit solche Daten vorhanden sind.

Es werden angemessene technische Maßnahmen eingesetzt. Eine absolute Sicherheit kann nicht garantiert werden.
