Discord Community Assistant

Magische Ordnung für deinen Server.

WitchKeeper vereint Verifizierung, Geburtstage, automatische Übersetzung sowie Willkommens- und Abschiedsmeldungen in einem multiserverfähigen Discord-Bot.

Funktionen ansehen GitHub
Funktionen

Ein Bot, viele Aufgaben

✅ Verifizierung

Mitglieder verifizieren sich per Button und erhalten automatisch die festgelegte Serverrolle.

🎂 Geburtstage

Geburtstage werden pro Server gespeichert. Am Geburtstag vergibt WitchKeeper eine Rolle und sendet Glückwünsche.

🌍 Übersetzung

Englischsprachige Nachrichten können automatisch ins Deutsche übersetzt werden.

🙌 Willkommen & Abschied

Neue und austretende Mitglieder werden automatisch im vorgesehenen Kanal angezeigt.

⚙️ Servereinstellungen

Administratoren verwalten Rollen, Kanäle und Funktionen über übersichtliche Slash-Befehle.

🖥️ Rund um die Uhr online

WitchKeeper läuft dauerhaft auf einem Linux-Server und ist unabhängig vom Entwickler-PC erreichbar.
